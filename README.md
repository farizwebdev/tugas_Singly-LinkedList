## Deskripsi
Program **Java** dan **Python** untuk mengelola data mahasiswa (NIM, Nama, Jenis Kelamin) menggunakan struktur data **Singly Linked List** secara manual (menggunakan Class Node sendiri, tanpa library `LinkedList` bawaan).

## Fitur
Program memiliki fitur CRUD lengkap dengan validasi data:
1. **Insert at beginning** (Tambah data di awal)
2. **Insert at given position** (Tambah data di posisi tertentu)
3. **Insert at end** (Tambah data di akhir)
4. **Delete from beginning** (Hapus data awal)
5. **Delete given position** (Hapus data posisi tertentu)
6. **Delete from end** (Hapus data akhir)
7. **Delete first occurence** (Hapus berdasarkan NIM)
8. **Show data** (Traversing / Menampilkan semua data)
9. **Exit**
